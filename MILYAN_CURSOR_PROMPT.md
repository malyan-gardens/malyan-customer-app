# مليان — Cursor Pro Prompt

## السياق
- **الشركة:** مليان للتجارة والحدائق — قطر
- **المالك:** زاهر توزري | zaher@malyangardens.com | GitHub: malyan-gardens
- **الخدمات:** نباتات صناعية + لاندسكيب + زراعة + صيانة
- **الفريق:** 5–10 فنيين وسائقين (قطر ودول أخرى)
- **الهدف:** تطبيق + موقع + لوحة تحكم داخلية + موظفين AI

---

## المشاريع الثلاثة (داخل فولدر مليان نفسه)

### 1. Dashboard (لوحة التحكم الداخلية)
- للإدارة وفاطمة (المسؤولة الرقمية)
- عرض: الطلبات، الفريق، الإحصائيات، المهام
- ألوان: أخضر غامق + ذهبي (هوية مليان)
- RTL، عربي، متجاوب

### 2. Driver App (تطبيق السائقين)
- للسائقين والفنيين في الميدان
- عرض: طلبات اليوم، المهام، العناوين، حالة التسليم/التنفيذ
- واجهة بسيطة للموبايل أولاً
- نفس الهوية البصرية

### 3. Customer App (تطبيق العملاء)
- للعملاء: عرض الخدمات، معرض النباتات، طلب استشارة/تواصل
- هومبج جاهز (تم بناؤه مسبقاً)
- يبقى داخل نفس الفولدر مليان
